## 🧬 DNA_BioinformaticsPortfolio

An interactive 3D bioinformatics portfolio that visualizes my academic and professional journey through a lab-inspired digital environment.

## 🌌 Overview

This project presents my work in bioinformatics, computational biology, and machine learning through an immersive web-based interface built using **Three.js**.

Instead of a traditional static resume, the portfolio simulates a virtual laboratory where users can explore my profile in an interactive way.

## 🖥️ How It Works
•On launch, users enter a **3D desktop laboratory environment** </br>
•Clicking the screen transitions into a **3D DNA helix visualization** </br>
•Each DNA node represents a different section of my profile  </br>

Users can:
•🦠 Hover over nodes to change cursor interaction
•🧬 Click nodes to open detailed information panels

Explore sections like: </br>
• About Me </br>
• Research Projects </br>
• Experience </br> 
• Technical Skills </br>
• Education  </br>
• Publications  </br>

## 🧪 Key Features
• Interactive **3D DNA helix navigation**  </br>
• Laboratory-themed UI/UX design  </br>
• Dynamic cursor switching (biohazard → DNA mode)  </br>
• Real-time hover and click interactions  </br>
• Immersive transition from desktop → molecular visualization  </br>
• Fully browser-based (no installation required)  </br>

## 🧠 Tech Stack
• Three.js (3D rendering)  </br>
• JavaScript (interaction logic)  </br>
• GSAP (animations)  </br>
• HTML5 / CSS3  </br>

## 🚀 Purpose

This project was created to transform a traditional CV into an **interactive scientific visualization experience,** reflecting my work in:

• Bioinformatics  </br>
• Machine Learning  </br>
• Multi-omics analysis  </br>
• Computational biology research  </br>
