## 🧬 DNA_BioinformaticsPortfolio

An interactive 3D bioinformatics portfolio that visualizes my academic and professional journey through a lab-inspired digital environment.

## 🌌 Overview

This project presents my work in bioinformatics, computational biology, and machine learning through an immersive web-based interface built using **Three.js**.

Instead of a traditional static resume, the portfolio simulates a virtual laboratory where users can explore my profile in an interactive way.

## 🖥️ How It Works
•On launch, users enter a **3D desktop laboratory environment**
•Clicking the screen transitions into a **3D DNA helix visualization**
•Each DNA node represents a different section of my profile

Users can:
•🦠 Hover over nodes to change cursor interaction
•🧬 Click nodes to open detailed information panels

Explore sections like:
•About Me
•Research Projects
•Experience
•Technical Skills
•Education
•Publications

## 🧪 Key Features
•Interactive **3D DNA helix navigation**
•Laboratory-themed UI/UX design
•Dynamic cursor switching (biohazard → DNA mode)
•Real-time hover and click interactions
•Immersive transition from desktop → molecular visualization
•Fully browser-based (no installation required)

## 🧠 Tech Stack
•Three.js (3D rendering)
•JavaScript (interaction logic)
•GSAP (animations)
•HTML5 / CSS3

## 🚀 Purpose

This project was created to transform a traditional CV into an **interactive scientific visualization experience,** reflecting my work in:

•Bioinformatics
•Machine Learning
•Multi-omics analysis
•Computational biology research
